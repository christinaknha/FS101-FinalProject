Agile Group Assignment

This is my favorite of all the projects that I have worked on so far. We were assigned to work with our accountability groups to develop an Ecommerce Website over the span of 2.5 days. This project had to have:

  • Navigation Bar and Footer on every page
  • A Homepage with a Hero, animation, and explanation/enticement for the user
  • 12 products on a Products Listing page
  • Detailed product pages for each product
  • Shopping Cart Prefilled
  • Checkout Page
  • Good web design principles

I enjoyed this project because my group made it such an enjoyable experience. We were able to collaborate and learn from each other and didn't have any conflicts. We had a lot of fun brainstorming product ideas and everyone did their share. It was a nice change from my past experiences with group work. I was in charge of the home page, shopping cart, putting together a CSS template, and formatting the product details pages. I really enjoyed the design aspects of the project. Bootstrap Assignment

This is one of my favorite projects that I have worked on so far. We were assigned to use VS Code to create a website of our choosing with set guidelines on what needed to be included. These guidelines included:

  • Navigation Bar
  • Hero with background image and call-to-action button
  • About Section with Text and Images
  • Features section with three columns using icons and text
  • Contact Section
  • Footer

I enjoyed this project because it was challenging to work with Bootstrap and I learned a lot that day. My biggest hurdle was making the page responsive to different sized screens and getting the cards spaced properly. I also struggled with getting the background image to fit properly onto the hero.
